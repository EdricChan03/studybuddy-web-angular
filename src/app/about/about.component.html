About Study Buddy

Study Buddy was started in 2017 as a personal project.

Who is Study Buddy for?

Study Buddy is primarily for students, although in the future, it may be extended to parents and teachers to faciliate the students.

Is this app open source?

Yes, it is! The source code for the Android app is available at Chan4077/StudyBuddy, while the source code for the web version is at Chan4077/StudyBuddy-web. You should be able to find instructions to build the source code in the associated GitHub repositories.

I would like to contribute to this project! How do I do so?

Thanks. I appreciate changes made by others. I accept PRs (pull requests) for those who actually use Github for open-source projects.
